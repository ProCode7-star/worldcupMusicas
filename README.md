# 🎵 Música del Mundial 2026 🏆

¡Bienvenido a mi proyecto web! Esta es una página creada completamente en **HTML** y **CSS** dedicada a recopilar las mejores canciones, himnos y ritmos oficiales (¡y no oficiales!) de la Copa Mundial de la FIFA 2026.

## 🚀 ¿Qué tiene esta página web?
- **Reproductor de Audio:** Para escuchar los mejores temas del mundial directo en la web.
- **Diseño Personalizado:** Un estilo limpio y moderno hecho desde un iPhone.
- **URL Pública:** Disponible para que cualquiera la pueda visitar en internet.

## 📱 Tecnologías usadas
* **HTML5:** Para estructurar toda la música, los títulos y los reproductores.
* **CSS3:** Para darle un diseño espectacular con colores y estilos llamativos.
